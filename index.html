Sobre Mim

Olá! Me chamo Felipe Bergamin Dantas, tenho 19 anos e sou apaixonado por tecnologia, dados e inovação. Atualmente, estudo Engenharia da Computação e trabalho com Análise de Dados, utilizando ferramentas como Power BI, SQL Server e Python. Tenho uma base sólida em lógica de programação e modelagem de dados, o que me permite transformar informações em insights valiosos. Sou movido por desafios, aprendizado constante e pela busca de soluções que gerem impacto real. Também tenho conhecimento em desenvolvimento web com HTML, CSS, JavaScript, React e TypeScript, o que amplia minha visão na hora de entender a jornada completa de uma aplicação do banco de dados à visualização final. Meu objetivo é crescer na área de dados, me tornar um futuro engenheiro de dados e contribuir para decisões mais inteligentes por meio da tecnologia.

Meus Projetos

Gerador de QR Code

O projeto consiste no desenvolvimento de um gerador de QR Codes simples e funcional. A ferramenta permite criar códigos personalizados a partir de links, textos ou outras informações inseridas pelo usuário. Seu objetivo é facilitar o compartilhamento rápido e seguro de dados em diferentes contextos.

Clone da página de vendas da Apple

Este projeto é um clone da página de vendas da Apple, desenvolvido com foco em design responsivo e fidelidade visual. O objetivo é reproduzir a interface original, destacando produtos, animações e elementos interativos. A iniciativa visa aprimorar habilidades em HTML, CSS e JavaScript, com atenção aos detalhes e à experiência do usuário.

Gerador de Tabuada

O projeto é um gerador de tabuada interativo, criado para facilitar o aprendizado da matemática de forma prática e dinâmica. O usuário pode inserir qualquer número e escolher até quantas vezes deseja multiplicá-lo. Desenvolvido com HTML, CSS e JavaScript, é uma ferramenta simples e educativa.

Calculadora de IMC

Este projeto é uma calculadora de IMC (Índice de Massa Corporal), desenvolvida para ajudar os usuários a verificarem sua condição física de forma rápida e prática. Basta inserir o peso e a altura para obter o resultado e sua classificação. A aplicação foi feita com HTML, CSS e JavaScript, com foco em acessibilidade e usabilidade.

Lazy Load

O projeto implementa a técnica de Lazy Load utilizando JavaScript puro, com o objetivo de otimizar o carregamento de imagens em páginas web. As imagens são carregadas apenas quando entram na área visível do usuário, melhorando o desempenho e a experiência de navegação. É uma solução leve e eficiente, focada em performance.

To-Do List

Este projeto é uma To-Do List interativa e completa, desenvolvida com HTML, CSS e JavaScript. Permite adicionar, editar, remover e pesquisar tarefas, além de filtrá-las entre concluídas, não concluídas e todas. Todos os dados são salvos no Local Storage, garantindo que as informações sejam mantidas mesmo após atualizar a página.A aplicação foi criada para ajudar na organização do dia a dia de forma simples e eficiente. Seu design é intuitivo e responsivo, focado na praticidade.
